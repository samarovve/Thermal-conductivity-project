# Thermal-conductivity-project


## Идея проекта

Проект посвящён численному моделированию тепловых процессов в кубе. Решаются две задачи:

1. **Стационарная задача** (часть 1)  
   Дан куб в тепловом равновесии. Известна температура сколь угодно многих точек на его границе. Требуется найти температуру в произвольной внутренней точке.  
   **Модель**: дискретное уравнение теплопроводности – температура каждого внутреннего элементарного кубика равна среднему арифметическому температур соседей. Получается система линейных уравнений, которая решается аналитически (символьное решение) или численно.

2. **Нестационарная задача** (часть 2)  
   Стенки куба теплоизолированы, внутри происходит кратковременный импульс тепла («взрыв»). Нужно найти зависимость температуры в заданной точке от времени.  
   **Модель**: гиперболическое уравнение теплопроводности с дискретизацией по пространству и интегрированием по времени:
   ```math
   \tau\frac{\partial^2T}{\partial t^2}+\frac{\partial T}{\partial t}=\alpha\nabla^2T+S(\mathbf{r},t)
   ```

Обе части будут сопровождаться визуализацией распределения температуры.
